How to Generate PNG Files

To generate the PNG files from these previews:

Method 1: Browser Screenshot

  1. Open this HTML file in a browser
  2. Right-click on each preview container
  3. Select "Inspect Element" or use browser DevTools
  4. Take a screenshot of each preview container at the exact dimensions
  5. Save as PNG files with the specified names

Method 2: Using ImageMagick

Create a script that renders these HTML previews to PNG using headless browser tools like Puppeteer or Playwright.

Method 3: Design Software

  1. Create new document at specified dimensions
  2. Apply brand gradient background
  3. Place logo SVG
  4. Add text with brand typography
  5. Export as PNG
