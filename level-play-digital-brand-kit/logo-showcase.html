Usage Guidelines

HTML Implementation

<!-- SVG Logo (Recommended) -->
<img src="01-logos/primary/level-play-digital-logo-gradient-full.svg" 
     alt="Level Play Digital" 
     width="300">

<!-- Inline SVG (for animations) -->
<svg xmlns="http://www.w3.org/2000/svg">
  <!-- SVG content -->
</svg>

CSS Styling

/* Logo sizing */
.logo {
  width: 200px;
  height: auto;
  max-width: 100%;
}

/* Logo on dark background */
.logo-dark-bg {
  filter: brightness(0) invert(1);
}

/* Logo hover effect */
.logo:hover {
  opacity: 0.8;
  transition: opacity 0.3s ease;
}

Minimum Sizes

/* Minimum recommended sizes */
Primary Logo: 120px width minimum
Icon Only: 48×48px minimum
Wordmark: 200px width minimum

/* Clear space */
Maintain clear space equal to height of "L" in "LEVEL" 
on all sides of the logo
