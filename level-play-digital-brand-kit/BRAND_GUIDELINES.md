# Level Play Digital Brand Guidelines

Complete brand identity guidelines for Level Play Digital. Follow these guidelines to ensure consistent brand representation across all applications.

## Brand Overview

Level Play Digital is a modern, forward-thinking brand that combines technical excellence with creative innovation. Our brand identity reflects this through a vibrant color palette, dynamic logo system, and professional design language.

## Logo System

### Primary Logo
The primary logo features the full "LEVEL PLAY DIGITAL" wordmark with the brand symbol, rendered in the signature gradient (cyan to orange at 135deg).

**Usage:**
- Primary brand representation
- Website headers
- Marketing materials
- Digital applications

**Requirements:**
- Minimum width: 120px
- Maintain aspect ratio
- Clear space: Height of "L" in "LEVEL" on all sides

### Logo Variations

**Solid Color Versions**
- Use when gradient is not feasible
- Print applications (use rich-black for professional print)
- Single-color applications
- Reverse applications (white on dark backgrounds)

**Icon Only**
- Favicons and app icons
- Small spaces (minimum 48×48px)
- Social media profile pictures
- When full logo doesn't fit

**Wordmark Only**
- Text-only applications
- Simplified branding
- When symbol doesn't fit context

### Logo Don'ts

❌ **Never:**
- Stretch or distort the logo
- Change colors or gradient direction
- Rotate the logo
- Place on busy backgrounds without clear space
- Use outdated versions
- Modify proportions
- Use gradient on small text

## Color System

### Primary Colors

**Cyan Blue (#00BFFF)**
- Primary brand color
- Use for: Links, CTAs, key brand elements
- RGB: rgb(0, 191, 255)
- HSL: hsl(193, 100%, 50%)
- CMYK: C:100 M:25 Y:0 K:0

**Coral Orange (#FF6633)**
- Secondary brand color
- Use for: Accents, secondary CTAs, complementary elements
- RGB: rgb(255, 102, 51)
- HSL: hsl(14, 100%, 60%)
- CMYK: C:0 M:60 Y:80 K:0

### Brand Gradient

**CRITICAL:** The gradient direction is fixed at 135deg (diagonal top-left to bottom-right). Never modify this direction.

- CSS: `linear-gradient(135deg, #00BFFF 0%, #FF6633 100%)`
- Use for: Logo, hero backgrounds, key visual elements
- Never use on small text (use solid colors instead)

### Neutral Colors

- **Dark**: #1a1a2e - Dark backgrounds, text on light
- **Dark Gray**: #2d2d44 - Secondary dark backgrounds
- **Medium Gray**: #6c757d - Secondary text, borders
- **Light Gray**: #e9ecef - Light backgrounds, subtle borders
- **Off White**: #f8f9fa - Page backgrounds, card backgrounds

## Typography

### Font Families

**Primary (Headings)**
- System fonts: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif
- Use for: Headings, navigation, emphasis

**Secondary (Body)**
- System fonts: Georgia, "Times New Roman", serif
- Use for: Body text, long-form content

### Type Scale

- H1: 2.5em / 1.2
- H2: 2em / 1.3
- H3: 1.75em / 1.4
- H4: 1.5em / 1.4
- H5: 1.25em / 1.5
- H6: 1em / 1.5
- Body: 1em / 1.6
- Small: 0.875em / 1.6

- XS: 4px
- SM: 8px
- MD: 16px

- XXL: 48px

## Usage Guidelines

### Digital Applications

**Websites**
- Use gradient logo for headers
- Maintain responsive sizing
- Ensure adequate contrast
- Use SVG format when possible

**Social Media**
- Use appropriate logo version for platform
- Maintain brand colors
- Follow platform-specific guidelines
- Use icon-only for profile pictures

**Email**
- Use inline CSS for styling
- Host logo online (don't embed)
- Maintain readability
- Test across email clients

### Print Applications

**Business Cards**
- Use rich-black version for professional print
- Maintain minimum sizes
- Ensure adequate bleed
- Use CMYK color mode

**Letterhead**
- Use appropriate logo version
- Maintain clear space
- Follow print specifications
- Use high-resolution files

## Brand Voice

### Tone
- Professional yet approachable
- Technical but accessible
- Innovative and forward-thinking
- Confident and clear

### Writing Style
- Clear and concise
- Active voice preferred
- Avoid jargon when possible
- Focus on benefits, not just features

## Accessibility

### Color Contrast
- Ensure minimum 4.5:1 contrast for normal text
- Ensure minimum 3:1 contrast for large text
- Test with color blindness simulators
- Provide alternative text for images

### Logo Accessibility
- Always include alt text
- Ensure sufficient clear space
- Maintain minimum sizes
- Test on various backgrounds

## Quality Assurance

Before using any brand asset, verify:

- [ ] Correct color values (#00BFFF, #FF6633)
- [ ] Gradient direction is 135deg
- [ ] Logo proportions are maintained
- [ ] Minimum sizes are met
- [ ] Clear space is adequate
- [ ] File format is appropriate
- [ ] Accessibility standards are met
