How to Generate PNG Files

To generate the PNG files from these previews, you can use one of the following methods:

Method 1: Browser Screenshot

  1. Open this HTML file in a browser
  2. Right-click on each preview container
  3. Select "Inspect Element" or use browser DevTools
  4. Take a screenshot of each preview container
  5. Save as PNG files with the specified names

Method 2: Using ImageMagick (Command Line)

  1. Install ImageMagick: brew install imagemagick (Mac) or download from imagemagick.org
  2. Convert SVG to PNG with background: 
    # Dark background
convert -background "#1a1a2e" -resize 1200x ../primary/level-play-digital-logo-gradient-full.svg level-play-digital-logo-on-dark-bg.png

# Light background  
convert -background "#ffffff" -resize 1200x ../primary/level-play-digital-logo-gradient-full.svg level-play-digital-logo-on-light-bg.png

# Transparent background
convert -background none -resize 1200x ../primary/level-play-digital-logo-gradient-full.svg level-play-digital-logo-transparent-bg.png

Method 3: Using Inkscape (GUI)

  1. Open the SVG file in Inkscape
  2. File → Document Properties → Set background color (or leave transparent)
  3. File → Export PNG Image
  4. Set width to 1200px, export
  5. Repeat for each background variation

Method 4: Using Node.js (sharp library)

Create a script to generate all PNG files programmatically. See generate-background-pngs.js for an example script.
