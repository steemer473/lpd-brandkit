# Level Play Digital Print Specifications

## Overview

This document outlines print specifications for Level Play Digital brand materials, ensuring consistent and high-quality printed outputs.

## Color Modes

### CMYK for Print
- **Cyan Blue**: C:100 M:25 Y:0 K:0
- **Coral Orange**: C:0 M:60 Y:80 K:0
- **Rich Black**: C:60 M:40 Y:40 K:100
- **White**: C:0 M:0 Y:0 K:0

### RGB to CMYK Conversion
- Always convert RGB files to CMYK before printing
- Use color profiles: ISO Coated v2 (ECI) or GRACoL
- Verify colors match brand standards

## Paper Specifications

### Business Cards
- **Size**: 3.5" x 2" (89mm x 51mm)
- **Finish**: Matte or Glossy
- **Weight**: 16pt (0.4mm) or 14pt (0.35mm)
- **Bleed**: 0.125" (3mm) on all sides
- **Safe Area**: 0.25" (6mm) from edges

### Letterhead
- **Size**: 8.5" x 11" (US Letter) or A4 (210mm x 297mm)
- **Finish**: Smooth, White
- **Weight**: 24lb (90gsm) or 28lb (105gsm)
- **Bleed**: Not required (full bleed not recommended)

### Envelopes
- **Size**: #10 (4.125" x 9.5") or DL (110mm x 220mm)
- **Finish**: Matching letterhead
- **Weight**: Matching letterhead

## Resolution Requirements

### Minimum Resolution
- **Business Cards**: 300 DPI
- **Letterhead**: 300 DPI
- **Large Format**: 150-200 DPI (depending on viewing distance)

### File Formats
- **Vector**: SVG, PDF (preferred for logos)
- **Raster**: TIFF, PNG (300 DPI minimum)
- **Final Output**: PDF/X-1a or PDF/X-4

## Bleed and Safe Areas

### Bleed
- **Standard**: 0.125" (3mm) on all sides
- **Purpose**: Ensures no white edges after trimming
- **Application**: Extend background colors/graphics to bleed area

### Safe Area
- **Standard**: 0.25" (6mm) from trim edge
- **Purpose**: Ensures important content isn't cut off
- **Application**: Keep text and logos within safe area

### Crop Marks
- Include crop marks for trimming
- Position at trim edge (not bleed edge)
- Use 0.5pt (0.18mm) lines in registration black

## Typography for Print

### Font Requirements
- Embed all fonts in PDF files
- Use system fonts or licensed fonts only
- Ensure fonts are print-ready (not web fonts)

### Minimum Font Sizes
- **Body Text**: 9pt minimum
- **Headings**: 12pt minimum
- **Fine Print**: 7pt minimum (use sparingly)

### Line Spacing
- **Body Text**: 1.5x font size
- **Headings**: 1.2x font size
- **Tight Spacing**: Avoid for readability

## Logo Usage in Print

### Minimum Sizes
- **Full Logo**: 1" width minimum
- **Icon Only**: 0.5" width minimum
- **Wordmark Only**: 0.75" width minimum

### Clear Space
- **Full Logo**: Equal to height of "LEVEL PLAY" text
- **Icon Only**: Equal to icon diameter
- **Wordmark Only**: Equal to height of text

### Background Requirements
- Use logo on white or light backgrounds
- Use white logo on dark backgrounds
- Use gradient logo on white backgrounds
- Never use logo on busy backgrounds

## Business Card Specifications

### Front Side
- **Logo**: Top left or center top
- **Company Name**: Below logo or prominent position
- **Tagline**: Optional, below company name
- **Contact Info**: Name, title, contact details

### Back Side
- **Logo**: Optional, smaller size
- **Additional Info**: Services, website, social media
- **QR Code**: Optional, for digital contact

### Layout Options
- **Vertical**: Standard orientation
- **Horizontal**: Alternative orientation
- **Two-sided**: Front and back designs

## Letterhead Specifications

### Header
- **Logo**: Top left corner
- **Company Name**: Below or beside logo
- **Contact Info**: Top right corner (optional)

### Footer
- **Logo**: Optional, smaller size
- **Legal Info**: Copyright, address, contact
- **Page Number**: If multi-page documents

### Content Area
- **Margins**: 1" (25mm) minimum on all sides
- **Line Spacing**: 1.5x for body text
- **Font**: Professional, readable serif or sans-serif

## File Preparation Checklist

Before sending files to print:

- [ ] Colors converted to CMYK
- [ ] Resolution is 300 DPI minimum
- [ ] Bleed added (0.125" / 3mm)
- [ ] Safe area respected (0.25" / 6mm)
- [ ] Crop marks included
- [ ] Fonts embedded
- [ ] File saved as PDF/X-1a or PDF/X-4
- [ ] Proofread all text
- [ ] Logo placement verified
- [ ] Brand colors verified

## Common Print Issues

### Color Mismatch
- **Issue**: Colors don't match screen
- **Solution**: Use CMYK color values, request color proof

### Font Issues
- **Issue**: Fonts not displaying correctly
- **Solution**: Embed fonts or convert to outlines

### Resolution Problems
- **Issue**: Images appear pixelated
- **Solution**: Ensure 300 DPI minimum, use vector logos

### Bleed Problems
- **Issue**: White edges after trimming
- **Solution**: Extend graphics to bleed area

## Printer Recommendations

### Digital Printing
- Suitable for small quantities (1-500)
- Faster turnaround
- Good quality for most applications

### Offset Printing
- Suitable for large quantities (500+)
- Better color accuracy
- Lower cost per unit for large runs

### Specialty Finishes
- **Foil Stamping**: For premium business cards
- **Embossing**: For textured logo
- **UV Coating**: For glossy finish
- **Die Cutting**: For custom shapes

## File Delivery

### Preferred Formats
- **PDF/X-1a**: Best compatibility
- **PDF/X-4**: Better color management
- **High-res TIFF**: For raster images

### File Naming
- Use descriptive names: `business-card-front.pdf`
- Include version numbers if needed
- Include date if multiple versions

### Delivery Method
- Email for small files (< 10MB)
- Cloud storage for large files
- FTP for very large files

## Quality Control

### Pre-Press Checklist
1. Verify all colors are CMYK
2. Check resolution is adequate
3. Confirm bleed and safe areas
4. Proofread all text
5. Verify logo placement and size
6. Check file format and compatibility

### Proof Review
1. Request digital proof first
2. Review color accuracy
3. Check text for errors
4. Verify layout and spacing
5. Approve before final print
