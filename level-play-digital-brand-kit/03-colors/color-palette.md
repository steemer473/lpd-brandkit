# Level Play Digital Color Palette

## Primary Brand Colors

### Cyan Blue (Primary)
- **HEX**: `#00BFFF`
- **RGB**: `rgb(0, 191, 255)`
- **HSL**: `hsl(193, 100%, 50%)`
- **CMYK**: `C:100 M:25 Y:0 K:0`
- **Usage**: Primary brand color, used for links, CTAs, and key brand elements

### Coral Orange (Secondary)
- **HEX**: `#FF6633`
- **RGB**: `rgb(255, 102, 51)`
- **HSL**: `hsl(14, 100%, 60%)`
- **CMYK**: `C:0 M:60 Y:80 K:0`
- **Usage**: Secondary brand color, used for accents and complementary elements

### Brand Gradient
- **CSS**: `linear-gradient(135deg, #00BFFF 0%, #FF6633 100%)`
- **Direction**: 135deg (diagonal top-left to bottom-right)
- **Usage**: Primary logo treatment, hero backgrounds, key visual elements
- **CRITICAL**: Never change gradient direction or color values

## Neutral Colors

### Dark (Charcoal Black)
- **HEX**: `#1a1a2e`
- **RGB**: `rgb(26, 26, 46)`
- **Usage**: Dark backgrounds, text on light backgrounds

### Dark Gray
- **HEX**: `#2d2d44`
- **RGB**: `rgb(45, 45, 68)`
- **Usage**: Secondary dark backgrounds, subtle elements

### Medium Gray
- **HEX**: `#6c757d`
- **RGB**: `rgb(108, 117, 125)`
- **Usage**: Secondary text, borders, dividers

### Light Gray
- **HEX**: `#e9ecef`
- **RGB**: `rgb(233, 236, 239)`
- **Usage**: Light backgrounds, subtle borders

### Off White
- **HEX**: `#f8f9fa`
- **RGB**: `rgb(248, 249, 250)`
- **Usage**: Page backgrounds, card backgrounds

## Semantic Colors

- **Usage**: Success messages, positive indicators

### Warning
- **HEX**: `#ffc107`
- **Usage**: Warning messages, caution indicators

### Error
- **HEX**: `#dc3545`
- **Usage**: Error messages, destructive actions

### Info
- **HEX**: `#17a2b8`
- **Usage**: Informational messages, neutral indicators

## Color Usage Guidelines

### Do's
- ✅ Use primary gradient for logo and key brand elements
- ✅ Use cyan blue for primary CTAs and links
- ✅ Use orange for accents and secondary CTAs
- ✅ Maintain 135deg gradient direction at all times
- ✅ Use exact HEX values (#00BFFF, #FF6633)

### Don'ts
- ❌ Never modify gradient direction
- ❌ Never use approximate colors
- ❌ Never use gradient on small text (use solid colors)
- ❌ Never use brand colors for error states (use semantic colors)

## Accessibility

### Contrast Ratios
- Primary blue on white: 3.4:1 (AA Large Text)
- Primary orange on white: 3.1:1 (AA Large Text)
- Dark text on off-white: 12.6:1 (AAA)
- White text on dark: 15.8:1 (AAA)

### Recommendations
- Use dark text on light backgrounds for body text
- Use white text on gradient backgrounds
- Ensure minimum 4.5:1 contrast for normal text
- Ensure minimum 3:1 contrast for large text
