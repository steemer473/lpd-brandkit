## Important Notes

### Print Requirements
- **Always convert RGB to CMYK** before printing
- Use PDF or SVG formats when possible (vector = scalable)
- PNG files are provided as fallback only
- Minimum resolution: 300 DPI for print

### Email Requirements
- Always embed images (host on your server)
- Include alt text for accessibility
- Keep total email size under 100KB when possible

### SMS Requirements
- Keep file sizes small (under 10KB preferred)
- Use icon-only versions when space is limited
- Test delivery across different carriers
